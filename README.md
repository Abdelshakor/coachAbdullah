# Dr. Abdallah — Online Coaching (Static Website)

## What is this?
A fast, responsive, bilingual (AR/EN) landing page designed to convert visitors into WhatsApp leads.

## Files
- index.html
- styles.css
- script.js
- assets/img/ (images)
- assets/favicon.svg

## Quick edit
- WhatsApp number is in `script.js` (WHATSAPP_NUMBER)
- Replace images in `assets/img/`
- Replace transformation placeholders in the "Transformations" section

## Run locally
Just open `index.html` in your browser (double-click).

## Deploy on GitHub Pages
1) Create a new repo (example: `dr-abdallah-coaching`)
2) Upload all files/folders (keep structure)
3) Repo Settings → Pages → Build and deployment:
   - Source: Deploy from a branch
   - Branch: main / root
4) Save, then open your site link.
